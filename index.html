CSS

Especificidad

La especificidad es la importancia que tiene un determinado selector, al cual se le asigna un valor determinado, podemos encontrar los siguientes valores:

  • Elemento html = 1
  • Clases = 10
  • id = 100
  • Inline= 1000 (En el html)
  • Important=infinito

    • Selectores

    los selectores son una forma para identificar los diversos elementos que se tiene en el html, dentro los cuales se encuentran :

  • Selector de etiqueta. ejemplo : body, head, div, a, H1, img, entre otras
  • Selector de clase. ejemplo : class="PRINCIPAL"
  • Selector de id. ejemplo id="IDENTI"

    • Cascada

    La cascada se aplica cuando se tiene la misma especificidad, en el cual se aplicaria el que aparece al ultimo ejemplo:

  • h3 { color: black; }
  • h3 { color: blue; } resultado color blue.

    • Herencia

    la herencia se da cuando un elemento padre transmite un propiedad css al hijo

    ejemplo: cuando se hereda del padre: p{ background: inherit; }
